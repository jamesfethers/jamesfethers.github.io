HTML Tags Cheatsheet

Tag Description Example
<Head> Metadata
<Body> Content rendered in the browser
<Div> Divide HTML doc into sections
<h> Heading level from 1-6
<li> Lists. Either ordered <ol> or unordered <ul>
<a> Anchor. Create hyperlinks to websites or documents using href <a href="http://www.codecademy.com">Visit this site</a>
<P> Paragraph element that displays blocks of text
<id> ID attributes differentiate between elements <id= "A1">
